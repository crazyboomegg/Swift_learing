## [02 - IndexedTable]
表格視圖製作索引列表
- 可以在標準表格視圖中顯⽰⼀系列的動物清單
- 除了列出所有動物外，這個 App 將動物群組起來至不同區塊，並顯⽰⼀個可以快速存取的索引列表。


<br/><br/>

## [03 - TableCellAnimation]
為表格視圖 cell 加入簡單的淡入動畫讓 App 更⽣動活潑
- 淡入動畫
- 使用 CATransform3D 建立旋轉特效
- 使用 CATransform3DTranslate 建立飛入特效
<br/><br/>

## [04 - KivaLoan]
呼叫Kiva API 來取得最近的融資貸款，並以表格視圖來做呈現

- 建⽴ JSON 資料模型
- 使⽤ Kiva API 取得貸款資料
- 使用 Codable 來解碼 JSON
- 在表格視圖顯示貸款資料
<br/><br/>
